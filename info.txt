# CROWDFUNDING PLATFORM - PROJECT DOCUMENTATION

## PROJECT TITLE
Crowdfunding Platform - A Full-Stack Web Application for Campaign Management and Donations

## PROJECT OBJECTIVES
1. Create a comprehensive crowdfunding platform where users can create, manage, and fund campaigns
2. Implement secure user authentication and authorization system
3. Provide real-time notifications and updates for campaign activities
4. Enable secure payment processing for donations
5. Develop an admin panel for platform management and analytics
6. Ensure responsive design for mobile and desktop compatibility
7. Implement data visualization for campaign analytics and reporting

## TECHNOLOGY STACK

### Frontend Technologies:
- **React.js 18.2.0** - Main frontend framework for building user interfaces
- **React Router DOM 6.8.1** - Client-side routing and navigation
- **Tailwind CSS 3.2.6** - Utility-first CSS framework for styling
- **Framer Motion 10.0.1** - Animation library for smooth transitions
- **Chart.js 4.2.1** - Data visualization and analytics charts
- **React Chart.js 2** - React wrapper for Chart.js integration
- **Axios 1.3.4** - HTTP client for API requests
- **React Hot Toast 2.4.0** - Toast notifications system
- **React Icons 4.7.1** - Icon library for UI components
- **Headless UI 1.7.12** - Unstyled, accessible UI components

### Backend Technologies:
- **Node.js** - JavaScript runtime environment
- **Express.js 4.18.2** - Web application framework
- **MongoDB** - NoSQL database for data storage
- **Mongoose 7.0.1** - MongoDB object modeling library
- **JWT (jsonwebtoken 9.0.0)** - Authentication and authorization
- **bcryptjs 2.4.3** - Password hashing and encryption
- **Multer 1.4.5** - File upload middleware
- **Cloudinary** - Cloud-based image and video management
- **Socket.io 4.6.1** - Real-time bidirectional communication
- **Stripe** - Payment processing integration
- **Razorpay** - Alternative payment gateway

### Security & Middleware:
- **Helmet 6.0.1** - Security headers middleware
- **CORS 2.8.5** - Cross-origin resource sharing
- **Express Rate Limit 6.7.0** - Rate limiting middleware
- **Express Mongo Sanitize 2.2.0** - MongoDB injection prevention
- **XSS Clean 0.1.1** - Cross-site scripting protection
- **HPP 0.2.3** - HTTP parameter pollution protection
- **Compression 1.7.4** - Response compression middleware

### Database Schema Design:

#### User Model:
- Personal information (name, email, phone)
- Authentication data (hashed password, JWT tokens)
- Profile settings (avatar, bio, social links)
- User statistics (campaigns created, donations made)
- Role-based permissions (user, admin)

#### Campaign Model:
- Campaign details (title, description, target amount)
- Media assets (images, videos)
- Campaign settings (deadline, category, tags)
- Financial tracking (current amount, donation count)
- Status management (pending, active, completed, cancelled)
- Updates and FAQs arrays

#### Donation Model:
- Donor and campaign references
- Payment information (amount, currency, payment method)
- Transaction status and timestamps
- Anonymous donation support

## API REFERENCES AND INTEGRATIONS

### Authentication APIs:
- **POST /api/auth/register** - User registration
- **POST /api/auth/login** - User login
- **POST /api/auth/forgot-password** - Password reset request
- **POST /api/auth/reset-password** - Password reset confirmation
- **GET /api/auth/me** - Get current user profile

### Campaign APIs:
- **GET /api/campaigns** - Get all campaigns with filtering
- **POST /api/campaigns** - Create new campaign
- **GET /api/campaigns/:id** - Get specific campaign details
- **PUT /api/campaigns/:id** - Update campaign
- **DELETE /api/campaigns/:id** - Delete campaign
- **POST /api/campaigns/:id/images** - Upload campaign images
- **POST /api/campaigns/:id/donate** - Make donation to campaign

### User Management APIs:
- **GET /api/users/profile** - Get user profile
- **PUT /api/users/profile** - Update user profile
- **POST /api/users/avatar** - Upload user avatar
- **GET /api/users/campaigns** - Get user's campaigns
- **GET /api/users/donations** - Get user's donation history

### Admin Panel APIs:
- **GET /api/admin/dashboard** - Admin dashboard statistics
- **GET /api/admin/users** - Get all users with pagination
- **PUT /api/admin/users/:id/role** - Update user role
- **GET /api/admin/campaigns** - Get all campaigns for review
- **PUT /api/admin/campaigns/:id/status** - Approve/reject campaigns
- **GET /api/admin/donations** - Get all donations with analytics

### Payment Integration:
- **Stripe Integration**: Credit/debit card processing
- **Razorpay Integration**: Alternative payment gateway for Indian market
- **UPI Support**: Direct UPI payments for Indian users
- **Payment Webhooks**: Real-time payment status updates

### Third-party Service Integrations:
- **Cloudinary**: Image and video upload, optimization, and delivery
- **Gmail SMTP**: Email notifications and password reset
- **Socket.io**: Real-time notifications and updates
